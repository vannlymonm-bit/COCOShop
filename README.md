# COCO Shop

โปรแกรมนี้เป็นโปรแกรมที่มีไว้จัดการสินค้าสำหรับร้านค้าที่ต้องการจัดสินค้าอย่างรวดเร็ว

- โปรแกรมที่ใช้เปิดไฟล์ Visual code studio 2026
- ภาษาที่ใช้ C#

## วิธีโหลด

1. เข้า github และเข้าสู่ระบบ(ถ้าไม่มีให้สมัคร)
2. เชื่อมไอดี github เข้ากับ visual code studio
3. กดปุ่ม <>code และกด open with visual studio
4. กด open visual studio

## วิธีการใช้งาน

1. เปิดโปรแกรม
2. คลิกที่ปุ่ม >Start
3. กรอกข้อมูล
4. กด ค้นหา

## ตัวอย่าง

1. ใส่ชื่อสินค้า เช่น หมวกกันน็อคKYT
2. เลือกประเภทสินค้า เช่น เครื่องส่วมใส่
3. ใส่ราคาและจำนวนสินค้า เช่น ราคา 3,500 จำนวน 1 เครื่อง
4. กดค้นหา ผลลัพธ์ แสดงใบเสร็จ
